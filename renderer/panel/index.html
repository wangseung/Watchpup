코드 레포 (원인 조사)

로컬·GitHub 레포를 Grep/Read로 뒤져 "코드 원인 조사"에서 원인·구현 위치를 찾습니다. GitHub는 한 번 클론해 로컬처럼 쓰고 개발→PR도 됩니다(gh 로그인 필요).

간편 연동 (Notion · Jira)

토큰만 넣으면 watchpup 전용 MCP로 자동 등록됩니다. 토큰은 Keychain에만 저장돼요(config엔 미저장).

📝 Notion

notion.so → 설정 → 연결 → 통합(Integration) 생성 후 발급. 해당 통합을 대상 페이지에 공유해야 읽힙니다.

🧩 Jira

id.atlassian.com → 보안 → API 토큰 생성.

MCP 연동 (분석 도구)

분석·워크플로우에 쓸 MCP 서버. 여기 등록한 것만 사용하며, 개인 Claude MCP 설정은 상속하지 않습니다.

학습한 교훈 (자가발전)

내 피드백·자가평가로 워크플로우별로 쌓여 다음 실행부터 자동 반영. 직접 추가·수정도 가능.
